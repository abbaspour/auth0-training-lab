Machine Setup
=============

Here is the list of required software packages:

- curl
- node.js
- git
- jq
- base64
- awk
- openssl


Mac
---
```bash
brew install curl node git jq coreutils openssl
```

Linux
-----

### Ubuntu/Debian
```bash
apt-get install curl node git jq openssl
```
